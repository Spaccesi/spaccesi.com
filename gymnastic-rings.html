Just a very basic model I did while I was learning how to use the features on Fusion360. You can see the model and some renders I did on that program.


Simulation


If you wish to 3D Print this model and use it you have to be careful, please remember safety first. As you will put your all bodyweight on it, the ring may fail and you may heart yourself. I only use them for make pull-ups, increasing the effort I must do during the exercise but not for more complex callisthenic exercises.
The following image shows the simulation of the model considering a apllied load of 20,000N and a Young's Module of 2,7GPa. As you can see the part is permanently deformated after such an effort.


3D Print


If you wish to 3D Print this model, I would recommend the following parameters:

  • Layer height: <0.2mm
  • Infill: 100%
